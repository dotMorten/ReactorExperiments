Simple file-based WinUI Reactor apps. See blog post: https://xaml.dev/post/file-based-winui-apps-with-microsoft-ui-reactor



To run simple app:



`dotnet run SimpleApp.cs -a x64`



To run optional-interactive mode app:

&#x20;- Console mode: `dotnet run InteractiveMode.cs -a x64`

&#x20;- Interactive (UI) mode: `dotnet run InteractiveMode.cs -a x64 -- -i`
